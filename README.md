# kakuji (書く字)

This project is an attempt to create a quiz for writing Japanese charcaters, namely Hiragana, Katakana, and Kanji.

Optimally, it will be easy to use on a mobile device for touch writing and easy submission - the goal is to write as many characters as possible, accurately, while simulatenously testing the user's knowledge of the readings, onyomi, kunyomi, and meanings. On desktop, easy key controls will be implemented for those who prefer to use another writing surface but want to be quizzed on different characters.
